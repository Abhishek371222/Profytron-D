# Motion Token Catalog

| Token | ms | Use |
|-------|----|-----|
| Instant | 0 | Press / snap |
| Fast | 120 | Hover, focus, toast backdrop |
| Standard | 200 | Modal, counter, toast |
| Slow | 320 | Navigation, drawer |
| Hero | 550 | Reserved (Phase 6 marketing) |
| Background | 800 | Ambient only |

Hover, Focus, Press, Success, Warning, Error, Loading, Refresh, Navigation, Modal, Toast, Tooltip, Drawer, Expansion, Collapse.
