# Premium Design Language

Every visual decision in the Experience layer reinforces at least one principle:

| Principle | Meaning |
|-----------|---------|
| **Confidence** | Bloomberg calm authority — no noisy chrome |
| **Precision** | TradingView clarity — tabular data, crisp edges |
| **Depth** | Layered surfaces (`--exp-depth-*`), not flat panels |
| **Calm** | Motion Engine budgets; no decorative storms |
| **Speed** | Progressive hero + perceived responsiveness |

Reject effects that do not support these principles.
