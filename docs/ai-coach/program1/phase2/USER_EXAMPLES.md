# User Examples

| User says | Expect |
| --- | --- |
| Why am I down today? | What/Why/Evidence/Meaning/Next + confidence + citations |
| Summarize this week | Period P&L / WR / strategy attribution if available |
| Explain my drawdown | Plain-language peak-to-trough + risk next step |
| Which strategy contributed most? | Best tagged strategy or honest “missing data” |
| Why was this trade opened? (with rail selection) | Stored explanation or explicit “no rationale” |

Never: autonomous close/pause; invented P&L; invented entry reasons.
