# Program 3 — Strategy Intelligence (Research)

**Status:** Complete  
**Depends on:** Program 2

## Job

Explain which strategies are working, struggling, improving, or degrading — with evidence.

## Locked intents

| Intent | Example |
|--------|---------|
| `strategy_compare_all` | Compare my strategies |
| `strategy_underperforming_why` | Why is this strategy underperforming? |
| `strategy_degraded` | Which strategy has degraded? |
| `strategy_improving` | Which strategy is improving? |
| `strategy_session_performance` | Win rate by session |
| `strategy_losing_symbols` | Losing symbols |
| `strategy_holding_time` | Holding time issues |

## Tools

`analytics_strategy_comparison`, `analytics_portfolio`, `analytics_trades`, `analytics_risk`, coaching report when present.
