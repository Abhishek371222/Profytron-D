# Program 5 — Performance Trend Intelligence (Research)

**Status:** Complete  
**Depends on:** Program 4

## Job

Explain whether performance is improving or declining using period metrics — not price prediction.

## Locked intents

| Intent | Example |
|--------|---------|
| `trend_improving_or_declining` | Am I improving or declining? |
| `trend_winrate_vs_avg_loss` | Win rate vs average loss |
| `trend_volatility_change` | Has volatility changed? |

## Tools

`analytics_portfolio` (multi-range when available), `analytics_risk`, `analytics_trades`, equity curve points.
