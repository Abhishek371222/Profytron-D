## Allowlist (Render API)

When list is final, set on **API** service env (comma-separated, no spaces preferred):

```text
BETA_ALLOWLIST_EMAILS=user1@example.com,user2@example.com
```

Empty allowlist = open registration — **do not leave empty** during closed beta.
