# Success Moments (A4)

Celebrate meaningful progress without noise.

| Moment | UX |
| --- | --- |
| Broker connected | Toast + checklist refresh · brief success panel in connect modal |
| First strategy activated | Existing confetti on deploy/subscribe |
| First completed trade | Achievement / notification (server) |
| First AI explanation viewed | Toast once: “First Coach insight unlocked” |
| Weekly portfolio summary available | Soft Coach suggestion (existing follow-ups) |

## Rules

- Once per user per milestone (activation events are unique).  
- No modal spam; prefer toast + checklist progress bump.  
- Never block the next action behind celebration UI.
