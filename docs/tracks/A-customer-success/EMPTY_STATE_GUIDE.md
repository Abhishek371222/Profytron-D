# Empty State Guide (A2)

Surface guidance where the user is — not a generic help center.

| Surface | Condition | Guidance |
| --- | --- | --- |
| Dashboard | No broker | Connect broker to see live account data |
| Strategies / My bots | No bots / no filters match | Create, import, or browse marketplace |
| Marketplace | Empty after filters | Reset filters · highlight beginner / featured |
| Marketplace | No broker yet | Connect account before subscribe |
| Alpha Coach first visit | Empty chat | Suggested starter questions · connect CTA if needed |
| Connected accounts | No accounts | Primary Connect CTA |

## Shared primitive

Prefer `DashboardEmptyState` + optional CTA slot (`actionHref` / `actionLabel`) for consistent activation-aware empties.
