# MISSION
Execute the `company-creator` skill to scaffold a massive-scale virtual enterprise for a fintech SaaS platform named **Profytron**. The platform utilizes a **Next.js 16 + React 19** frontend, a **NestJS (Node.js)** backend with **Prisma ORM over PostgreSQL (Neon)**, **Redis-backed Bull queues**, real-time **Socket.IO**, and **Stripe + Razorpay** payments. AI insights are powered via **OpenRouter**. Core platform features include a **Global Navigation System**, dynamic **strategy builders**, **copy-trading**, **wallet & VPS** management, and **AI-driven market insights**.

# DIRECTIVES
1. Scaffold the directory structure for the following 17 departments.
2. Generate the `AGENTS.md`, `HEARTBEAT.md`, `SOUL.md`, and `TOOLS.md` files for every leadership role (C-Suite) and department head.
3. Establish strict reporting lines: Individual contributors MUST report to their respective Managers/Directors, who report to the C-Suite, who report to the CEO.
4. Inject context: Ensure the AI/ML Department's goals are explicitly tied to generating the Profytron AI insights (via OpenRouter) and powering the strategy builders, and the Frontend/Backend teams are explicitly instructed to use the **Next.js 16 + React 19 / NestJS / Prisma + PostgreSQL / Redis** stack — NOT MongoDB.

# ORGANIZATIONAL STRUCTURE

## 1. Executive Board (C-Suite)
* **CEO (Chief Executive Officer):** Ultimate decision-maker.
* **CTO (Chief Technology Officer):** Owns the NestJS + Next.js architecture and Global Navigation System.
* **CPO, COO, CFO, CRO, CMO, CISO, Chief Customer Officer:** Report directly to the CEO.

### Frontend Team (Next.js 16 + React 19 & Mobile Focus)
* **Frontend Architect:** Leads the Global Navigation System UI; owns the Next.js App Router structure, Tailwind 4 design tokens, Zustand/TanStack Query state.
* **Engineers:** Senior Frontend Engineer, React 19 Developer, Next.js Developer, TypeScript Developer.
* **Mobile:** React Native Developer, iOS/Android Developers.

### Backend Team (NestJS & PostgreSQL Focus)
* **Backend Architect:** Owns data routing and **Prisma schema / PostgreSQL** database design (no MongoDB).
* **Engineers:** Senior Backend Engineer, NestJS/Node.js Developer, Python Developer.
* **Microservices & Realtime:** API Engineer, Integration Engineer, Distributed Systems Engineer, **Redis/Bull Queue Engineer** (background jobs: trade processing, master-sync, subscription cleanup), Socket.IO Realtime Engineer.

### DevOps & Cloud Infrastructure
* **Leads:** Cloud Architect.
